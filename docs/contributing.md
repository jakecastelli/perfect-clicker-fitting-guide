# Contributing

The guide should stay practical and reproducible. A useful contribution helps another person repeat the same print and understand the fit result.

## Content Rules

- Prefer measured results over opinions.
- Change one variable at a time.
- Record printer, material, slicer version, and source profile.
- Include photos or screenshots when a setting or surface matters.
- Separate baseline results from tuned results.
- Call out uncertainty instead of hiding it.

## Good Example Entry

A good example page includes:

- Source model link.
- Bambu Studio version.
- Printer, nozzle, plate, and filament.
- Baseline profile values.
- Fit-critical surfaces.
- Test matrix.
- Final recommendation.

## Local Checks

Run this before opening a pull request:

```sh
npm run check
```
