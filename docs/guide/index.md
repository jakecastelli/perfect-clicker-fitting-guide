# Guide Overview

This guide is for printed clickers where the important outcome is fit and feel: the parts should assemble cleanly, move freely, click consistently, and avoid sloppy looseness.

The first example is the MakerWorld [Capybara Clicker Keychain](https://makerworld.com/en/models/3029730-capybara-clicker-keychain#profileId-3405004). Exact screenshots and slicer values should be added after opening the model in Bambu Studio.

## What Good Fit Means

A good clicker fit is not just "the parts go together." Record these outcomes for every test print:

| Outcome | Good result |
| --- | --- |
| Assembly | Parts seat without cracking, shaving, or forced pressure. |
| Movement | The click area moves without binding through the full travel, and the top cap does not touch or scratch the base when pressed down. |
| Return | The mechanism returns consistently after each press. |
| Sound | The click is repeatable, not mushy or muted. |
| Looseness | No obvious rattle unless the model intentionally needs free play. |
| Finish | No required cleanup beyond normal brim/support removal. |

## First Content Milestones

1. Capture the MakerWorld model profile and Bambu Studio version.
2. Print a baseline using the downloaded profile without fit tweaks.
3. Identify the actual fit surfaces in the model.
4. Build a small iteration table from the first three test prints.
5. Add screenshots so the guide is reproducible by a new user.
